# ArkIDE Website and Backend API
A rewritten version of the website with a backend API and new astro language!

## API Uses:
- Serves data for the downloads, news, and update image data
- Helps keep the main website code clean from 90 line long json strings
- Makes adding a new version much simpler and a new renderer simpler too!
## Homepage Uses:
- Its a homepage.
- Takes all of the data from the API and hosts it onto itself via the download manager and news pages etc.
